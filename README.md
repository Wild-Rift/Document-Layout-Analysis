# Document-Layout-Analysis
Tools for extract figure, table, text,... from a pdf document

The directories should be arranged like this

    root
    ├── mmdet
    ├── tools
    ├── configs
    ├── datasets
    │   ├── publaynet
    │   │   ├── test
    │   │   ├── train
    │   │   ├── val
    │   │   ├── train.json
    |   |   ├── val.json
